# video-transcript-mindmap

A Codex skill for turning one or more video URLs into a deliverable package with:

- a complete transcript
- a structured mind map
- optional report outputs in Markdown, Word, PowerPoint, PDF, and zip form
- optional long-form professional analysis reports for teaching, review, or archival use

## Included

- `SKILL.md` - main skill instructions
- `evals/` - evaluation assets for the skill

## Notes

This local repository was split out so the skill can evolve independently from unrelated workspace repositories.
